# Neovim shortcuts
**yy**         - copy line

**dd**         - cut line

**p**          - paste

**yw**         - copy word

**v[←↑→↓]y**   - copy range

**v[←↑→↓]d**   - cut range

**u**          - undo

**[space]e**   - NvimTreeToggle

**[space]b**   - new buffer / tab

**[space]x**   - close buffer / tab

**[tab]**      - switch tabs

**[space]/**   - toggle comment

**a**          - new file

**d**          - rm file

**[ctrl]w↑**   - navigate to up window

**[ctrl]w↓**   - navigate to down window

**[ctrl]w←**   - navigate to left window

**[ctrl]w→**   - navigate to right window

**[ctrl]\n**   - unfocus terminal window
